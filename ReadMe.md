**Consigne d'installation :**  
---  
> Télécharger ou clone le gitlab  
  
## Windows :  
> Installer python3 : https://www.python.org/downloads/  
> Aller dans le dossier PyreStone  
> Ouvrir menu.py  

## Linux ( Ubuntu ) :   
> `sudo apt-get install python3`  
> `sudo apt install python3-pil python3-pil.imagetk python3-tk python-imaging-tk`  
> `cd lien/vers/le/dossier/PyreStone`  
> `python3 menu.py`  

## Mac :  
> `brew install python3`  
> `curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py`  
> `python3 get-pip.py`  
> `python3 -m pip install pip`  
> `python3 -m pip install Pillow`  
> `cd lien/vers/le/dossier/PyreStone`  
> `python3 menu.py`
